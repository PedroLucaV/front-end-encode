Oque é o HTML?

É uma linguagem de marcação usada na construção de uma pagina web. Documentos HTML são interpretados por todos os navegadores

Oque é uma linguagem de marcação?

É simplesmente uma forma de simbolizar oque cada coisa é através de "tags" que estão entre <>, exemplo: p, h1, h2, ul, ol, li

Site que usa somente HTML

Ao entrar no wayback machine podemos entrar no google com somente o HTML, o qual antigamente não usava CSS

Estruturas básicas de uma pagina web

Aprender essas principais tags que são necessários para que o documento seja corretamente interpretado pelos navegadores

  • !DOCTYPE html: Informa ao navegador que a partir daqui será um código HTML
  • html: É a tag que delimita até onde vai o código html, nesse caso ela é obrigatória para abrir e fechar a tag com a barra
  • head: Representa o cabeçalho do documento da aplicação e sinaliza todos os metadados, informando todas as informações que são de configurações da página
  • body: Representa o corpo e a parte visível da aplicação web

Formatando nossos textos

O texto mais estruturado é composto por títulos e parágrafos, independentemente de estar lendo uma historia, um jornal, um livro de texto da faculdade, uma revista, etc

Como formatar

Para formatar iremos usar duas tags iniciais, uma que representa o titulo "<h1>" e o paragrafo "<p>", sem eles ele não irá entender a formatação.

Exemplos:

Este é o paragrafo

Texto em Itálico, Negrito e Sublinhado

Para deixar um texto em itálico usamos a tag <em>texto</em>

Para um texto em Negrito usamos o <strong> texto </strong>

Para um texto Sublinhado usamos o <u> texto </u>

Texto com Errado, Sobrescrito e Subscrito

Para um texto Errado usamos o <del> texto </del>

Para um texto Sobrescrito usamos o <sup> texto </sup>

Para um texto Subscrito usamos o <sub> texto </sub>

Texto pequeno, Destacado e

Para um texto pequeno usamos o <small> texto </small>

Para um texto Errado usamos o <mark> texto </mark>

AVISO!

Essas formatações servem somente para serem usadas como forma semântica e não visual, ou seja, ela precisa de um sentido no texto para que seja aplicada
