Como a internet funciona?

Os computadores chamados de "Servidores" servem para disponibilizar o serviço da internet.

Como as informações são distribuídas?

A internet pega informações do Servidor para distribui-las para o Cliente, o qual são pedidas pelo próprio cliente

Como funciona essa troca?

Ao ter a requisição do usuário para tal informação do servidor, o cliente irá puxar do provedor de internet, o qual olhará a identificação através do DNS(Domain Name Server, identificador do servidor requisitado), desse forma voltando para nós o que pesquisamos

Como um website funciona?

Um website é composto com a seguinte estrutura:

  • Uma linguagem de marcação: "HTML"
  • Uma linguagem de estilização: "CSS"
  • Uma linguagem de programação: "JavaScript"

Qual a diferença?

  • HTML: É a estruturação do website, textos, organização etc
  • CSS: É a estilização do website: Cores, Design, Shape etc
  • JavaScript: São as ações realizadas pelo site
